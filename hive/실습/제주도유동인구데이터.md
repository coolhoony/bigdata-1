##Step 2: 위경도좌표를 이용하여 14시의 제주 유동인구를 add 단위로 쿼리
```
select t2.*,t3.* from (
select c.add,sum(t.FLOW_POP_CNT_14TMST) from pnu c , jeju t where c.code  = substr(t.pnu,0,10)
group by c.add
  ) t2,latitude t3 where t2.add = t3.add;
```
